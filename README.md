# Interview-Sales-Project---Power-BI
-This project presents an interactive E-Commerce Sales Dashboard developed using Power BI to analyze and visualize sales performance. The dashboard provides insights into key business metrics such as total sales, profit, quantity sold, and customer purchasing trends.

-The dataset was imported from Microsoft Excel and transformed using Power Query for data cleaning, formatting, and preparation. Data modeling techniques were applied to establish relationships between tables and create calculated measures to enhance analytical capabilities.

-The dashboard includes various visualizations such as bar charts, pie charts, line charts, and KPI cards to help users quickly understand sales trends and performance across different categories, regions, and time periods.

🔧 Tools & Technologies Used
-Power BI Desktop
-Microsoft Excel (Data Source)
-Power Query (ETL – Extract, Transform, Load)
-Data Modeling
-DAX Measures

Interactive Visualizations
-📈 Key Features
-Sales performance overview with KPIs
-Category-wise and region-wise sales analysis
-Profit and quantity tracking
-Time-based sales trends
-Interactive filters and slicers for better exploration

🎯 Project Objective
-The goal of this project is to demonstrate how business intelligence tools can convert raw sales data into meaningful insights that support data-driven decision making.
